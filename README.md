📊 Student Performance Analytics & Prediction System
End-to-end Random Forest ML pipeline with Power BI dashboard to identify at-risk students (84% accuracy)
 using real educational data.

🚀 Project Objective

To analyze how factors such as attendance, participation, and resource usage impact student grades and to build a predictive model that identifies students at academic risk.

Analytics	Pandas, NumPy, Matplotlib
